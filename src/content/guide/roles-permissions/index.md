## Team Collaboration

Youth Coach Hub supports multi-coach collaboration with different permission levels. Each team member has a role that determines what they can see and do within your team.

### What You'll Learn

- **[Owner / Head Coach](/guide/roles-permissions/owner-head-coach)** - Full control over team and billing
- **[Assistant Coach](/guide/roles-permissions/assistant-coach)** - Coaching access without admin controls
- **[Managing Team Members](/guide/roles-permissions/managing-team-members)** - Adding, removing, and changing roles

### Role Overview

| Role | Description |
|------|-------------|
| **Owner** | Team creator with full control and billing access |
| **Coach** | Full coaching features without admin/billing access |
| **Analyst** | Film tagging and analytics access |
| **Viewer** | Read-only access to team content |

### Quick Reference

**Who can...**

| Action | Owner | Coach | Analyst | Viewer |
|--------|-------|-------|---------|--------|
| View playbook | ✓ | ✓ | ✓ | ✓ |
| View analytics | ✓ | ✓ | ✓ | ✓ |
| Tag plays | ✓ | ✓ | ✓ | ✗ |
| Create plays | ✓ | ✓ | ✓ | ✗ |
| Manage roster | ✓ | ✓ | ✗ | ✗ |
| Manage schedule | ✓ | ✓ | ✗ | ✗ |
| Upload film | ✓ | ✓ | ✗ | ✗ |
| Invite coaches | ✓ | ✗ | ✗ | ✗ |
| Manage billing | ✓ | ✗ | ✗ | ✗ |
| Delete team | ✓ | ✗ | ✗ | ✗ |

### Choosing the Right Role

- **Owner** - The head coach or primary administrator
- **Coach** - Assistant coaches and coordinators
- **Analyst** - Volunteers helping with film review
- **Viewer** - Parents, players, or observers
