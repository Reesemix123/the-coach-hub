## Overview

The Game Report provides a comprehensive breakdown of a single game, including scoring summary, offensive and defensive stats, and key metrics compared to your season averages.

## Accessing the Report

1. Navigate to **Analytics** in your team dashboard
2. Select **Game Report** from the report selector
3. **Select a game** from the game filter dropdown

> **Note:** You must select a specific game to view this report.

---

## Game Summary

The top section shows:

- **Final Score** - Your score vs. opponent
- **Quarter-by-Quarter Breakdown** - Scoring by period
- **Game Result** - Win, Loss, or Tie
- **Date and Opponent** - Game identification

---

## Offensive Section

### Volume Stats

| Stat | Description |
|------|-------------|
| **Total Yards** | Combined rushing and passing |
| **Rushing Yards** | Ground game production |
| **Passing Yards** | Aerial production |
| **Plays Run** | Total offensive snaps |

### Scoring Breakdown

| Stat | Description |
|------|-------------|
| **Touchdowns** | Total TDs scored |
| **Field Goals** | FGs made |
| **Extra Points** | PATs converted |
| **2-Point Conversions** | Two-point plays made |
| **Total Points** | Points from tagged plays |

### Efficiency

| Stat | Description |
|------|-------------|
| **Yards Per Play** | Average gain |
| **3rd Down Conversions** | Third downs converted |
| **Turnovers** | Giveaways in this game |

### Penalty Impact

| Stat | Description |
|------|-------------|
| **Penalties (Us)** | Flags against your team |
| **Penalty Yards (Us)** | Yards lost to penalties |
| **Penalties (Opponent)** | Flags on opponent |
| **Penalty Yards (Opponent)** | Yards gained from opponent penalties |

## Comparison to Season Averages

Each stat shows how this game compares to your season averages:

- **↑ Green** - Better than average
- **↓ Red** - Worse than average
- **→ Gray** - Near average

This helps you identify whether this was an above-average or below-average performance.

---

## Using the Game Report

### Post-Game Review

After each game, use the Game Report to:

1. Review final stats
2. Identify what went well
3. Find areas that struggled
4. Compare to expectations

### Game Planning

When preparing for a similar opponent:

1. Find games against similar teams
2. Review what worked
3. Note what didn't work
4. Build your game plan accordingly

### Player Meetings

Use specific game stats to:

- Celebrate achievements
- Identify improvement areas
- Set goals for next game

---

## Tips

### Tag Accurately

Game Report accuracy depends on:

- All plays being tagged
- Correct yardage entered
- Proper play type attribution

### Review Multiple Games

Use Game Report for several games to identify patterns and trends.
