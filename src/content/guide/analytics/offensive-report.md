## Overview

The Offensive Report provides detailed analysis of your offense, including team stats, individual player performance by position, formation effectiveness, and play concept breakdowns.

## Accessing the Report

1. Navigate to **Analytics** in your team dashboard
2. Select **Offensive Report** from the report selector
3. Use filters to analyze all games or specific games

---

## Team Offensive Stats

### Overall Production

| Metric | Description |
|--------|-------------|
| **Total Plays** | Offensive snaps |
| **Total Yards** | Combined rushing + passing |
| **Yards Per Play** | Average gain per snap |
| **Success Rate** | % of plays achieving goals |
| **Touchdowns** | Total TDs scored |

### Run Game

| Metric | Description |
|--------|-------------|
| **Rush Attempts** | Total carries |
| **Rush Yards** | Ground yardage |
| **Yards Per Carry** | Average per rush |
| **Rush Success Rate** | % of successful runs |
| **Explosive Runs** | 10+ yard carries |

### Pass Game

| Metric | Description |
|--------|-------------|
| **Pass Attempts** | Total passes |
| **Completions** | Passes caught |
| **Pass Yards** | Aerial yardage |
| **Yards Per Attempt** | Average per pass |
| **Pass Success Rate** | % of successful passes |
| **Explosive Passes** | 15+ yard completions |

### Run/Pass Balance

Visual breakdown showing:

- Percentage of runs vs. passes
- Success rate comparison
- Yards per play comparison

## Using the Report

- Struggling position groups

- Which concepts to emphasize
- Which players to feature
- Formation tendencies to consider
