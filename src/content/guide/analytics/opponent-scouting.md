## Overview

The Opponent Scouting report provides comprehensive analysis of your opponents based on film you've tagged. Understand their defensive tendencies, offensive patterns, and special teams approach to prepare effective game plans.

## Accessing the Report

1. Navigate to **Analytics** in your team dashboard
2. Select **Opponent Scouting** from the report selector (under Scouting)
3. Choose an opponent from the dropdown
4. View defensive, offensive, and special teams tabs

---

## Prerequisites

To see opponent tendencies, you must first tag opponent plays in Film:

1. Go to **Film** and select a game
2. When tagging plays, check the **"Opponent Play"** checkbox
3. Tag enough plays to build confidence (30+ plays recommended)

The more plays you tag, the more accurate the analysis becomes.

## Defensive Tendencies

Understand what defenses you'll face:

### Overall Metrics

| Metric | What It Shows |
|--------|---------------|
| **Blitz Rate** | How often they bring extra rushers |
| **Run Stop Rate** | Effectiveness against the run |
| **Pass Defense Rate** | Effectiveness against the pass |

### Coverage Distribution

See which coverages they use most:

- Cover 3, Cover 2, Cover 1, Cover 0
- Man vs Zone percentages
- Helps plan route concepts

### High Blitz Situations

Know when to expect pressure:

- 3rd down blitz rates
- Red zone blitz rates
- Passing down tendencies

### Recommendations

Based on tendencies, get suggestions like:

- "Use quick passes and screens to exploit blitz-heavy approach"
- "Target the seams against Cover 3"
- "Establish the run - opponent struggles to stop ground game"

### Run/Pass Split

| Metric | What It Shows |
|--------|---------------|
| **Run Rate** | Percentage of run plays |
| **Pass Rate** | Percentage of pass plays |
| **Yards Per Run** | Average gain on run plays |
| **Yards Per Pass** | Average gain on pass plays |

See which formations they use most:

- Top formations by usage percentage
- Run rate from each formation
- Pass rate from each formation

### Concept Frequencies

Identify their favorite plays:

- Top run concepts (Power, Zone, Counter, etc.)
- Top pass concepts (Levels, Curl-Flat, etc.)

## Special Teams Tendencies

Prepare for all phases:

### Kickoff & Punt Tab

| Metric | What It Shows |
|--------|---------------|
| **Kickoff Distance** | Average kickoff length |
| **Touchback Rate** | Percentage that reach end zone |
| **Onside Attempt Rate** | Watch for surprises |
| **Punt Distance** | Average punt length |
| **Hang Time** | How long punts stay airborne |
| **Inside 20 Rate** | How often they pin you deep |
| **Fake Attempt Rate** | Watch for trick plays |

### Returning Tab

| Metric | What It Shows |
|--------|---------------|
| **Kick Return Avg** | Yards per kick return |
| **Punt Return Avg** | Yards per punt return |
| **Fair Catch Rate** | How aggressive the returner is |
| **Return TD Rate** | Scoring threat level |

### Scoring Tab (FG/PAT)

| Metric | What It Shows |
|--------|---------------|
| **FG Accuracy** | Overall field goal percentage |
| **Accuracy by Range** | Success at different distances |
| **PAT Accuracy** | Extra point reliability |
| **2-Point Rate** | How often they go for two |

### Key Insights

Get actionable intelligence:

- "Weak punter - expect good field position"
- "Dangerous kick returner - directional kick"
- "Watch for fake FGs (X% attempt rate)"

| Under 30 plays | Low confidence - use with caution |

Tag more plays to improve analysis accuracy.

## Using the Report

### Pre-Game Preparation

1. Review opponent tendencies before game week
2. Identify their strengths and weaknesses
3. Build a game plan that exploits weaknesses
4. Prepare players for what they'll see

### Practice Planning

Use tendencies to guide practice:

- Run scout team with opponent's formations
- Practice against their favorite plays
- Prepare for high-blitz situations

### In-Game Adjustments

Reference during the game:

- "They run 60% of the time from Shotgun"
- "Expect blitz on 3rd and long"
- "Their kicker struggles from 40+"

## Building Your Scouting Database

### Upload Opponent Film

For best results, upload dedicated scouting film:

1. Go to **Film**
2. Click **Add Opponent Game**
3. Enter opponent name and upload their game film
4. Tag their plays (with "Opponent Play" checked)

### Tag Regular Games

You can also analyze opponents from games you've played:

1. Tag your offensive plays normally
2. Also tag opponent defensive plays (check "Opponent Play")
3. Both contribute to opponent tendencies

### Combine Sources

Mix both sources for comprehensive analysis:

- Game film shows how they played against you
- Scouting film shows their tendencies overall
