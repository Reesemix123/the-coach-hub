## Overview

The Situational Report breaks down your performance in specific game situations. Understand how you perform on critical downs, in different field positions, and under various game conditions.

## Accessing the Report

1. Navigate to **Analytics** in your team dashboard
2. Select **Situational Report** from the report selector
3. Use filters to analyze all games or specific games

---

## Down and Distance

### First Down Performance

| Metric | Description |
|--------|-------------|
| **Success Rate** | % gaining 4+ yards |
| **Average Gain** | Yards per 1st down play |

| **Explosive Rate** | Big plays on 1st down |

## Using the Report

### Identify Problem Areas

Find situations where you struggle:

- Which downs give you trouble?
- Where on the field do you struggle?
- Game situations that hurt you?

### Build on Strengths

Recognize where you excel:

- Best down-and-distance situations
- Strongest field position zones
- Game situations where you thrive

### Game Planning

Use situational data to:

- Prepare for opponent tendencies
- Script opening drives
- Plan red zone packages
- Develop 3rd down packages

### In-Game Adjustments

When reviewing situational data:

- What worked last time in this situation?
- What's our best play on 3rd & 5?
- What do we run when we need to protect a lead?
