## Overview

The Defensive Report provides detailed analysis of your defense, including team stats, individual player performance by position group, and opponent tendencies.

## Accessing the Report

1. Navigate to **Analytics** in your team dashboard
2. Select **Defensive Report** from the report selector
3. Use filters to analyze all games or specific games

---

## Team Defensive Stats

### Overall Performance

| Metric | Description |
|--------|-------------|
| **Plays Defended** | Defensive snaps |
| **Total Yards Allowed** | Combined rush + pass allowed |
| **Yards Per Play Allowed** | Average gain allowed |
| **Opponent Success Rate** | % of opponent plays successful |
| **Points Allowed** | Total scoring defense |

### Run Defense

| Metric | Description |
|--------|-------------|
| **Rush Attempts Faced** | Opponent carries |
| **Rush Yards Allowed** | Ground yardage given up |
| **Yards Per Carry Allowed** | Average per opponent rush |
| **Rush Success Rate Allowed** | % of successful opponent runs |

### Pass Defense

| Metric | Description |
|--------|-------------|
| **Pass Attempts Faced** | Opponent passes |
| **Completions Allowed** | Catches given up |
| **Pass Yards Allowed** | Aerial yards allowed |
| **Yards Per Attempt Allowed** | Average per opponent pass |

## Using the Report

### Identify Strengths

Find what your defense does best:

- Best players by position
- Strongest situational areas
- High-impact playmakers

### Address Weaknesses

Spot areas needing work:

- Positions with low production
- Situational struggles (3rd down, red zone)
- Opponent tendencies exploiting you

### Scheme Adjustments

Use opponent data to:

- Anticipate formations
- Prepare for common concepts
- Adjust coverage schemes
